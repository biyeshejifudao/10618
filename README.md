# 基于vue的个人课表管理

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=10618

## 技术介绍
Java语言 + ssm框架/springboot框架（后端）+ Vue前端框架（Web）+ Mysql(数据库)

## 开发工具
IDEA或者MyEclipse + Navicat(mysql数据库可视化工具) + Tomcat

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/828%E5%A5%97ssm%E5%BD%95%E5%83%8F/10618_ssm461%E5%9F%BA%E4%BA%8Evue%E7%9A%84%E4%B8%AA%E4%BA%BA%E8%AF%BE%E8%A1%A8%E7%AE%A1%E7%90%86%2Bvue%E5%BD%95%E5%83%8F.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
